# bak-cli — Brand & Visual Identity

## Product Context

**Name**: bak-cli
**Binary**: `bak`
**Tagline**: Pack your AI coding setup. Move anywhere.
**Purpose**: CLI tool that backs up, restores, and syncs AI coding configurations (OpenCode) across machines.
**Language**: Go
**Target audience**: Developers using AI coding tools (OpenCode, Cursor, Claude Code, etc.)

## Brand Personality

- **Professional** but approachable
- **Technical** but not intimidating
- **Reliable** — emphasizes safety and data integrity
- **Modern** — clean, minimal, developer-friendly

## Color Palette (suggestions)

- Primary: Deep blue (#1a1a2e) or dark teal (#0f3460)
- Accent: Electric blue (#00d4ff) or cyan (#16c79a)
- Background: Dark (#0d1117) — GitHub dark mode compatible
- Text: White (#ffffff) or light gray (#c9d1d9)

## Logo Concepts (pick one or combine)

### Concept 1: The Backpack
- Minimalist backpack icon made of geometric shapes
- Straps form a subtle "b" letter
- Clean lines, no gradients

### Concept 2: The Shield + Box
- Box/package icon with a small shield overlay
- Represents "backup" + "security"
- Shield has a checkmark

### Concept 3: The Terminal Prompt
- `bak>` terminal prompt as the main element
- Monospace font, blinking cursor
- Surrounded by subtle file/folder icons

### Concept 4: Abstract "bak" Letters
- The letters "bak" in a custom monospace font
- Each letter has a subtle icon embedded:
  - b = box (backup)
  - a = arrow (sync)
  - k = key (security)
- Minimal, typographic

## Banner Requirements

- **Size**: 1280×640px (GitHub social preview)
- **Content**: Logo + tagline + one-line description
- **Style**: Dark background, clean typography
- **Text**:
  - Main: `bak`
  - Sub: `Pack your AI coding setup. Move anywhere.`
  - Small: `CLI tool for backing up OpenCode configurations`

## What to Generate

1. **Logo** — Square icon (512×512px), works on dark and light backgrounds
2. **Banner** — Social preview image (1280×640px) for GitHub
3. **Favicon** — 32×32px version of the logo

## Prompt for Image Generation

```
Create a minimalist, modern logo for a developer CLI tool called "bak". 
The tool backs up AI coding configurations across machines.

Style: Clean, geometric, monoline. Dark background (#0d1117). 
Accent color: electric blue (#00d4ff) or cyan (#16c79a).

The logo should work as both a square icon and a horizontal banner.
No gradients, no 3D effects, no photorealism.
Think: Vercel logo, Linear logo, Railway logo — minimal and confident.

For the banner: include the text "bak" in large monospace font, 
with tagline "Pack your AI coding setup. Move anywhere." below.
Add small text: "CLI tool for backing up OpenCode configurations"
```
